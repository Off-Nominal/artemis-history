# Artemis IV - First Crewed Lunar Landing

## Designated: February 27, 2026

Under the restructured Artemis manifest, Artemis IV is now the first
crewed lunar landing mission.

## Mission Profile

- First humans on the Moon since Apollo 17 (December 1972)
- First mission to the Lunar Gateway
- Delivers the I-Hab (International Habitation Module) to Gateway
- Crew descends to the lunar south pole via Human Landing System
- Target: Early 2028

## Architecture
- SLS Block 1B with Exploration Upper Stage
- Orion with European Service Module
- Lunar Gateway (PPE + HALO, pre-positioned)
- I-Hab module (co-manifested)
- Human Landing System (SpaceX Starship HLS)
