# Space Launch System (SLS)

## Overview
The Space Launch System is NASA's next-generation heavy-lift launch vehicle,
authorized by the NASA Authorization Act of 2010.

## Design Heritage
- Core stage powered by four RS-25 engines (Space Shuttle Main Engines)
- Twin 5-segment solid rocket boosters (derived from Shuttle SRBs)
- Built at NASA's Michoud Assembly Facility in New Orleans

## Configurations

### Block 1
- LEO payload: ~70 metric tons (154,000 lbs)
- Trans-lunar injection: ~27 metric tons
- Upper stage: Interim Cryogenic Propulsion Stage (ICPS)

### Block 1B (planned)
- Upgraded Exploration Upper Stage (EUS)
- LEO payload: ~105 metric tons

### Block 2 (planned)
- Advanced boosters
- LEO payload: ~130 metric tons
- Most powerful rocket ever built

## Target First Launch
2017 (Exploration Mission-1)
