# Space Launch System (SLS)

## Design Unveiled: September 14, 2011

NASA officially unveiled the Space Launch System design, a Shuttle-derived
heavy-lift rocket intended to be the most powerful launch vehicle ever built.

## Core Stage
- Length: 212 feet (64.6 m)
- Diameter: 27.6 feet (8.4 m)
- Propellant: Liquid hydrogen and liquid oxygen
- Engines: Four RS-25D/E (Aerojet Rocketdyne)
- Thrust: ~1.6 million lbs (sea level, all 4 engines)

## Solid Rocket Boosters
- Two 5-segment solid rocket boosters (Northrop Grumman)
- Thrust: ~3.6 million lbs each
- Derived from Space Shuttle SRBs with added fifth segment

## Configurations

### Block 1
- LEO payload: ~70 metric tons
- Upper stage: Interim Cryogenic Propulsion Stage (ICPS) by ULA
- First mission: Exploration Mission-1 (uncrewed lunar flyby)

### Block 1B (planned)
- Exploration Upper Stage (EUS) with four RL-10 engines
- LEO payload: ~105 metric tons

### Block 2 (planned)
- Advanced boosters (type TBD)
- LEO payload: ~130 metric tons

## Manufacturing
- Core stage: Michoud Assembly Facility, New Orleans, LA
- Assembly: Vehicle Assembly Building, Kennedy Space Center, FL
- Testing: Stennis Space Center, MS

## Target First Launch
2017 (Exploration Mission-1)
