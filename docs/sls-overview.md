# Space Launch System (SLS)

## Current Status (2017)

SLS development is progressing but behind the original 2017 first-launch
target. First flight now expected no earlier than 2019-2020.

## Key Hardware Milestones
- RS-25 engine test firings ongoing at Stennis Space Center (2015-2017)
- ICPS (Interim Cryogenic Propulsion Stage) delivered by ULA to KSC (July 2017)
- Core stage structural test articles under testing
- Core stage production underway at Michoud Assembly Facility since Nov 2014

## Core Stage
- Length: 212 feet (64.6 m)
- Diameter: 27.6 feet (8.4 m)
- Propellant: Liquid hydrogen and liquid oxygen
- Engines: Four RS-25D/E (Aerojet Rocketdyne)
- Thrust: ~1.6 million lbs (sea level, all 4 engines)

## Solid Rocket Boosters
- Two 5-segment solid rocket boosters (Northrop Grumman)
- Thrust: ~3.6 million lbs each

## Configurations

### Block 1
- LEO payload: ~70 metric tons
- Upper stage: ICPS by ULA (delivered to KSC July 2017)
- First mission: Exploration Mission-1 (uncrewed)

### Block 1B (planned)
- Exploration Upper Stage (EUS)
- LEO payload: ~105 metric tons

### Block 2 (planned)
- Advanced boosters
- LEO payload: ~130 metric tons

## Manufacturing
- Core stage: Michoud Assembly Facility, New Orleans, LA
- Assembly: Vehicle Assembly Building, Kennedy Space Center, FL
- Testing: Stennis Space Center, MS
