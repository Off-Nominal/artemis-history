# SLS / Artemis Launch Schedule History

This document tracks the evolution of launch date targets over time.

## Exploration Mission-1 (later Artemis I)

### August 7, 2014 - Key Decision Point C (KDP-C)
SLS Block 1 passes KDP-C, formally entering full-scale development.

- **Launch target**: November 2018
- **Internal target**: December 2017
- **Cost commitment**: ~$9.7 billion for SLS development

This establishes the official baseline from which all future schedule
slips will be measured. NASA committed to this date before Congress.
