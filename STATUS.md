# Space Launch System / Orion — Program Status

> As of January 20, 2017

**NASA Administrator:** Robert M. Lightfoot Jr. (Acting Administrator)
**Goal:** Moon vicinity (proving ground) → cislunar space → Mars (mid-2030s)
**Next Milestone:** SLS core stage production; EM-1 first flight (November 2018 target)

## Vehicles
- **Space Launch System (SLS): Passed KDP-C — full development authorized, EM-1 targeted November 2018**
- **Orion MPCV: EFT-1 complete — 4.5-hour uncrewed test flight successful**

## Overview
Robert Lightfoot is serving as Acting Administrator following Bolden's departure. The Trump administration is expected to shift NASA's priorities away from the asteroid mission and back toward the Moon.
