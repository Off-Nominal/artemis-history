# Space Launch System / Orion — Program Status

> As of March 25, 2015

**NASA Administrator:** Charles F. Bolden Jr. (12th Administrator)
**Goal:** Asteroid Redirect Mission (retrieve boulder from large asteroid); Mars in the 2030s
**Next Milestone:** SLS core stage production; EM-1 first flight (November 2018 target)

## Vehicles
- **Space Launch System (SLS): Passed KDP-C — full development authorized, EM-1 targeted November 2018**
- **Orion MPCV: EFT-1 complete — 4.5-hour uncrewed test flight successful**

## Overview
NASA has selected ARM Option B: a robotic spacecraft will retrieve a multi-ton boulder from a large near-Earth asteroid and redirect it into lunar orbit for crewed study. This option demonstrates planetary defense techniques. SLS and Orion continue toward EM-1.
