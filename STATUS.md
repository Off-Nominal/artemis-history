# Artemis — Program Status

> As of December 9, 2020

**NASA Administrator:** Jim Bridenstine (13th Administrator)
**Goal:** Land Americans on the Moon by 2024
**Next Milestone:** SLS Green Run hot fire; HLS provider down-select

## Vehicles
- **Space Launch System (SLS): Core stage at Stennis for Green Run — 8-test series**
- **Orion MPCV: EFT-1 complete — 4.5-hour uncrewed test flight successful**
- **Human Landing System: Solicitation issued (NextSTEP-2 Appendix H)**
- **Lunar Gateway: Removed from Artemis III critical path — not required for first landing**

## International Partners
Artemis Accords: 8 founding nations (U.S., Australia, Canada, Italy, Japan, Luxembourg, UAE, UK)

## Overview
NASA has named 18 astronauts eligible for Artemis lunar missions. SLS Green Run testing continues at Stennis, with the hot fire expected in early 2021.
